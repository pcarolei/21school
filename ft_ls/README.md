#	ft_ls
My implementation of the ls command written in C.

###	Requirements
-   GNU make
-	GCC compiler

###	Building
1.	git clone https://github.com/pcarolei/21school
2.	cd 21school/ft_ls
3.	make

###	Running
./ft_ls <flags>

###	Supported flags
*	-l	(list in long format)
*	-a	(display all files)
*	-t	(sort by modification date)
*	-r	(reverse sort)
*	-R	(recursively list subdirectories)
